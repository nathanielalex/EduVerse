Sorting Algorithm

Sorting algorithms are used to rearrange an array or list of given elements according to the comparison operator on the elements. The comparison operator is used to determine the new order of elements in a data structure.

Selection Sort

Selection sort adalah salah satu dari algoritma sorting yang menemukan elemen terkecil dalam list dan menempatkannya di sisi paling kiri. Untuk melakukan ini, ia menggunakan strategi untuk membandingkan elemen pertama dari unsorted list dengan sisa list dan menukar posisi dengan elemen yang lebih kecil.

S[i], S[smallest] = S[smallest], S[i] means the position of the ith element and the position of the element with the smallest value are swapped.
